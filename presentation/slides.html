Biggest Takeaways

  • GenAI tools are bending how we build and deliver software
  • Aspects of waterfall (like planning/requirements) are still important
  • Moved from one-shot prompting to structured approaches
  • Eyes opened to Spec Driven Development (SDD)

Key Discoveries

  • Inner vs Outer Agents - different tools for different purposes
  • Memory management - critical for context and continuity
  • Chat/Agent modes - novel and practical way to get different outputs
  • SDD works with any AI tooling - vendor agnostic approach

Brownfield Apps

1000% these are great tools for brownfield applications
  • Documentation & onboarding generation is effortless
  • The human becomes the bottleneck - reviewing and digesting output
  • Contributed to modern .NET codebase with zero prior .NET experience

"Will we get to a point where we're using AI for everything and we're just the humans reviewing and approving?"

Beyond Feature Development

  • Gap Analysis - comparing current state vs desired state
  • Compliance Planning - helped create FedRAMP compliance roadmap
  • GitHub Issues - automated issue creation via tooling
  • Using AI right and left of just software development

Practical Advice

  • Use these tools on projects - and outside of projects
  • Stay up to date with the latest and greatest
  • Adapt to client openness - AI-forward vs conservative approaches
  • Trust matters - when contributing without deep domain knowledge
